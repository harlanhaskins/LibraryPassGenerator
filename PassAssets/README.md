# Pass Assets

Place your pass images in this directory. The following files are supported:

## Required
- **icon.png** (29x29 pt) - Icon shown on lock screen and in Wallet
- **icon@2x.png** (58x58 px)
- **logo.png** - Logo shown in top-left of the pass

## Optional
- **icon@3x.png** (87x87 px)
- **logo@2x.png**
- **logo@3x.png**
- **background.png** - Background image for the pass
- **background@2x.png**
- **background@3x.png**
- **strip.png** - Strip image (event tickets, coupons)
- **strip@2x.png**
- **strip@3x.png**
- **thumbnail.png** - Thumbnail image (event tickets, boarding passes)
- **thumbnail@2x.png**
- **thumbnail@3x.png**

All files must be PNG format. The @2x versions are for retina displays, @3x for higher resolution displays.

Note: Points (pt) vs Pixels (px) - Use @2x for 2x pixel density and @3x for 3x pixel density.
